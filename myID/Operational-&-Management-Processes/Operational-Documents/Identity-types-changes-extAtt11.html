External attribute 11 changes depending on Identity types 

 

  • If identitytype is Personal Primary and myIDPersonEmployeeGroup contains “789” then extAtt11 will be updated to external-main else internal-main
  • If identitytype is Personal Secondary and owner’s myIDPersonEmployeeGroup contains “789” then extAtt11 will be updated to external-additional else internal-additional
  • If identitytype is Shared ID, department
  • If identitytype is System ID, system-id
  • If identitytype is Robot ID, robot-id
  • If identitytype is Shared Mailbox, department-ln
  • If identitytype is null, extAtt11 will be updated to null
